## 自动配置

- 先运行`1配置后端.bat`

- 再运行Html页面



## 手动配置

- 先配置好ollama环境变量:

```
OLLAMA_HOST: :11434
OLLAMA_ORIGINS: *
```

- 重启ollama

- 在浏览器中打开html
